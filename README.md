# 👋 สวัสดีGitHub
## ข้อมูลของฉัน
- ชื่อ-สกุล :นายกันทรากร ดำนิ่ม
- ชื่อเล่น :ทิว
- สาขาวิชา :วิทยาการคอมพิวเตอร์และสารสนเทศ วิทยาการดิจิทัล
## เป้าหมายในการเรียนรายวิชานี้
ทักษะและความรู้ใหม่ๆ เพื่อนำไปพัฒนาตัวเองในอนาคต
## สิ่งที่ฉันสนใจ
- AI
- Game Development
- Game Design
- Novel

## ความคาดหวังจากการเรียนรายวิชานี้

1. ได้เรียนรู้การใช้ Git และ GitHub
2. สามารถพัฒนาเว็บไซต์ได้ด้วยตัวอง
3. เพิ่มทักษะในการพัฒนาเว็บไซต์ได้
4. ความเข้าใจด้านเว็บไซต์เพิ่มมากขึ้น
